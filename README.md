# Dream Theater — Symbolic Pass

A premium symbolic ticket pass for gifting.

## Customize
Use URL params:

- name
- month
- year
- city
- venue
- giftedBy

Example:
`?name=Juan%20Perez&month=April&year=2026&city=Buenos%20Aires&venue=Movistar%20Arena&giftedBy=Fernando`

## Deploy
- GitHub Pages: Settings → Pages → Deploy from branch → root
- Or Netlify / Vercel as static site

## Cómo subirlo a GitHub Pages (en 60 segundos)

1. Subís el repo a GitHub
2. GitHub → Settings → Pages
3. Source: Deploy from branch
4. Branch: main / root
5. Te da un link tipo `https://tuusuario.github.io/dream-theater-gift-ticket/`

¡Y listo!
